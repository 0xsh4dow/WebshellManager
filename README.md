## WebshellManager

> 一款无聊的时候用PHP写的一句话WEB端管理工具 名字叫w8ay好了

## 介绍

- 目前仅支持对PHP的一句话进行操作
- 暂未完成用户登陆
- 完成文件管理，支持对webshell中的文件进行增、删、改、查操作 可多选操作
